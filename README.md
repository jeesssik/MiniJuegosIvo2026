# MiniJuegosIvo2026

## DroneScape: Escapa del Firewall

DroneScape es un minijuego web tipo puzzle ambientado en un entorno tecnológico. En este juego, un drone de seguridad debe escapar de un laberinto digital (firewall), y el jugador lo guía programando una secuencia de movimientos con flechas.

La propuesta combina una estética oscura y temática de ciberseguridad con conceptos educativos de pensamiento computacional.

## Objetivo del juego

Guiar el drone desde su posición inicial hasta la salida del laberinto sin chocar contra los muros del firewall.

Para lograrlo, el jugador debe:

- Planificar una ruta antes de ejecutar.
- Construir una secuencia correcta de comandos (↑ ↓ ← →).
- Corregir errores cuando haya colisiones o rutas incompletas.

## Funcionalidad principal

- Niveles progresivos con laberintos de distinta dificultad.
- Controles por botones y también por teclado (flechas).
- Sistema de secuencia programada:
	- Agregar comandos.
	- Deshacer último comando.
	- Reiniciar nivel.
	- Ejecutar secuencia completa.
- Feedback visual del recorrido:
	- Camino visitado.
	- Colisión con firewall.
	- Éxito al alcanzar la salida.
- Pantalla de victoria con avance al siguiente nivel.

## Competencias y conceptos trabajados

Este minijuego está diseñado para reforzar:

- Algoritmos: definir pasos ordenados para resolver un problema.
- Secuencias: construir y ejecutar instrucciones en orden correcto.
- Planificación: anticipar movimientos y optimizar la ruta.
- Pensamiento lógico: detectar errores y ajustar estrategias.

## Estética y temática

- Ambiente escolar con narrativa de misión digital.
- Paleta visual oscura (estilo laboratorio/sistema de seguridad).
- Interfaz clara para aprender jugando.

## Tecnologías usadas

- HTML5
- CSS3
- JavaScript (vanilla)
